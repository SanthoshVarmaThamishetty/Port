## 🤖 About This Portfolio

A **cyberpunk-themed, interactive single-page portfolio** built with pure HTML, CSS, and vanilla JavaScript — zero frameworks, zero dependencies. Features a drag-to-reveal robot intro animation that connects wires to tech nodes before launching the main portfolio.

> **4.0/4.0 GPA** · Applied AI · George Brown College · 1.5+ Years Experience

---

## ✨ Features

- 🎮 **Interactive Intro Animation** — Drag the robot to the center to connect all tech nodes and launch the portfolio
- ⚡ **Animated Wire Connections** — Bezier-curved cables spark to life as the robot moves
- 🌀 **Orbiting Tech Icons** — Five tech badges orbit the hero robot in continuous animation
- 📊 **Animated Skill Bars** — Triggered on scroll via IntersectionObserver
- 🎞 **Ticker Tape** — Auto-scrolling tech stack marquee
- 📱 **Fully Responsive** — Mobile-first, touch drag supported
- 🚫 **Zero Dependencies** — Pure HTML/CSS/JS, no frameworks or build tools
